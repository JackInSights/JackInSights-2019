JackInSights - 2019

Annual Statistical Breakdown

Inspired by Daniel Hritzkiv's Annual Report series, JackInSights - 2019 is an exercise in personal data collection and reflection.


I had previously mapped out this data using Microsoft Excel spreadsheets but with the advent of tracking applications this is significantly easier. This website will be static but I hope as I learn to code, I hope to use API calls to directly access the data from the tracking apps.


I started tracking myself again late 2019 so this data will not be complete. (Late October 2019)

My goals for 2020 is to track in more detail using various apps and try my best to be consistant for the entire year.
