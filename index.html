JackInSights - 2019

Annual Statistical Breakdown

Inspired by Daniel Hritzkiv's Annual Report series, JackInSights - 2019 is an exercise in personal data collection and reflection.


I had previously mapped out this data using Microsoft Excel spreadsheets but with the advent of tracking applicaitons this is significantly easier. This website will be static but I hope as I learn to code I will be able to make it more interactable


I started tracking myself again late 2019 so this data will not be complete.

My goals for 2020 is to track in more detail using various apps and try my best to be consistant for the entire year.
